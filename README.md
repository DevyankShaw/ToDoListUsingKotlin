# ToDoListUsingKotlin
A Todo list app where user can add, edit, delete and update tasks and sub tasks. Sub tasks can also be dragged up and down

## Skills
Android, Kotlin and SQLite

## Widgets
Recycler View, Navigation Drawer, Floating Action Bar
